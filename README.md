Govhack Perth 2016 Entry
=============

https://govhack-perth-2016.herokuapp.com/

## To Install

`npm install`

`npm install -g tsd`

`tsd install`


## To Run (Dev)

`gulp`

open browser to http://localhost:3000

## To Build Release

`gulp release`

## To Publish to Heroku

`git push heroku master`
